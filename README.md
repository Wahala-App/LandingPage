# Wahala Landing Page

A premium, high-performance landing page for the Wahala App.

## Features
- **Modern UI**: Glassmorphism, Neon Gradients, Dark Mode.
- **Smooth Animations**: Powered by Framer Motion.
- **Responsive**: Fully responsive flex/grid layouts.
- **Components**:
  - Hero Section (Animated entrance)
  - Contact Team (Holographic cards)
  - Feature Demo (Glass dashboard mockup)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` to view.

## Technologies
- React 18
- Vite
- Framer Motion
- Lucide React
- CSS Modules / Variables
